# Images escalade

Photos d'action en paroi : grimpe en tête, second en mouvement, départ de longueur.

**Format recommandé** : WebP, max 1200px de large, qualité 85
**Nommage suggéré** : `escalade-[sujet]-[numero].webp`
Exemples : `escalade-leader-en-dalle-01.webp`, `escalade-clippage-01.webp`

**Usage dans le texte** :
```md
![Un grimpeur en dalle](/images/escalade/escalade-dalle-01.webp)
```
