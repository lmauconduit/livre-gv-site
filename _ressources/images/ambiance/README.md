# Images ambiance

Photos de paysage, parois, ambiance montagne : lever de soleil sur la falaise,
cordée vue de loin, paysage du Verdon, voie au coucher du soleil...

**Format recommandé** : WebP, max 1600px de large, qualité 85
**Nommage suggéré** : `ambiance-[lieu]-[sujet]-[numero].webp`
Exemples : `ambiance-verdon-falaise-01.webp`, `ambiance-cordee-vue-loin-01.webp`

**Usage dans le texte** :
```md
![Vue sur le Verdon](/images/ambiance/ambiance-verdon-01.webp)
```
